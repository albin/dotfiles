# dotfiles

## Configuration

### bash
`config/.bash_aliases`

### tmux
`.tmux.conf`

### vim
`config/.vim/`  
`config/.vim/.vimrc`

### Alacritty
`config/.alacritty.yml`

### X
`config/.Xresources`

### PuTTY
`config/putty_colors_default_settings.reg` - replaces colors for default settings

## Setup

### Linux
`# ./setup.sh`

### Linux (manual)
1. Install all packages
2. Link all files
3. Create a backup folder for vim (`~/.vim/backup`)
